Prompt in. Pictures out.

Image Studio is a tiny native Mac app that turns one sentence into a batch of images. It rides your ChatGPT login, costs nothing extra, and drops every result into a folder you own.

Download for macOS View on GitHub

macOS 15+ · 2.4 MB · free and open source

Watch a batch land

One prompt, four parallel requests. Each image drops into the gallery the moment it finishes.

Free with your ChatGPT subscription

The app reuses your local codex login. No new key, no new bill. Prefer a relay instead? Any OpenAI-Images-compatible endpoint works, and the per-image price is shown before you spend.

Every image gets its own lane

One request per image, all in parallel. A failed slot retries by itself. You can send the next batch while this one is still cooking.

Your folder is the history

Results land in a folder you pick. Press Space to Quick Look, drag one to Finder, right-click to use it as a reference. No database, no account, no lock-in.

No placebo controls

Size options match what each backend actually accepts, verified against live endpoints. Anything the Codex channel cannot do honestly routes to the relay.

Running in three minutes

1

Install

git clone github.com/Suge8/Image-Studio cd Image-Studio && make install
2

Connect

Run codex login once in Terminal, or paste a relay base URL and API key in Settings.

3

Generate

Write a prompt, press ⌘↩. Finished images stream into the gallery one by one.
